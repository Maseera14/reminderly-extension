# Reminderly — Railway Deployment

Railway is a platform that supports persistent Node.js servers with Puppeteer/Chromium.

## Steps to Deploy on Railway

1. Go to https://railway.app and sign up with GitHub
2. Click "New Project" → "Deploy from GitHub repo"
3. Select: Maseera14/reminderly-extension
4. Railway auto-detects Node.js and deploys

## Environment

- PORT is set automatically by Railway
- Data is persisted on Railway's disk
